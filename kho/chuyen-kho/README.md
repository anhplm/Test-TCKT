# Chuyển kho

Mô tả nghiệp vụ chuyển kho thường phát sinh trong doanh nghiệp.

Các nghiệp vụ chuyển kho cơ bản:

* Chuyển vật tư hàng hóa giữa các kho nội bộ.
* Chuyển hàng gửi bán đại lý.
